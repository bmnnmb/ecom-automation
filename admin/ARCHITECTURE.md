# 电商运营管理系统架构设计 v2.0

## 1. 设计理念

### 1.1 核心原则
- **数据驱动**：所有决策基于实时数据，提供数据洞察
- **用户中心**：围绕电商运营人员工作流设计，提升工作效率
- **模块化**：各功能模块独立，可按需组合和扩展
- **响应式**：支持PC、平板、手机等多终端访问

### 1.2 设计参考
- **抖店后台**：实时数据看板、短视频/直播整合
- **有赞后台**：私域运营、会员体系、营销自动化
- **微盟后台**：全渠道整合、智慧零售、数据中台
- **CRMEB风格**：左侧分组菜单+顶部Header+灰白底色+蓝色主色调

## 2. 系统架构

### 2.1 前端技术栈
- **框架**：React 18 + TypeScript
- **状态管理**：Redux Toolkit + RTK Query
- **路由**：React Router 6
- **UI组件库**：Ant Design 5（定制CRMEB风格主题）
- **图表库**：AntV G2/G6（数据可视化）
- **构建工具**：Vite 5
- **样式方案**：CSS Modules + Less

### 2.2 后端集成
- **API网关**：localhost:8000（统一入口）
- **微服务**：
  - 抖音适配器：localhost:8001
  - 拼多多适配器：localhost:8002
  - 闲鱼适配器：localhost:8003
  - 快手适配器：localhost:8004
  - OMS服务：localhost:8005
  - RAG服务：localhost:8009
  - 竞品爬虫：localhost:8007
  - 控制中心：localhost:8008

### 2.3 数据流
```
用户操作 → 前端组件 → Redux Action → RTK Query → API网关 → 微服务 → 数据库
```

## 3. 功能模块设计

### 3.1 一级菜单结构
1. **工作台**（Dashboard）
2. **商品中心**（Products）
3. **订单中心**（Orders）
4. **客户中心**（Customers）
5. **营销中心**（Marketing）
6. **数据中心**（Analytics）
7. **客服中心**（Service）
8. **财务中心**（Finance）
9. **供应链**（Supply Chain）
10. **系统管理**（System）

### 3.2 详细模块设计

#### 3.2.1 工作台（Dashboard）
**核心功能**：
- 实时数据看板（今日/本周/本月核心指标）
- 趋势图表（营收、订单、客户、商品趋势）
- 异常预警（库存预警、退款预警、差评预警）
- 待办工作流（待发货、待处理退款、待回复消息）
- 平台对比（多平台经营数据对比）
- 快捷入口（常用功能快速访问）

**数据指标**：
- 营收总额、净利润、利润率
- 订单量、客单价、转化率
- 新客户数、复购率、客户价值
- 商品销量、库存周转、动销率

#### 3.2.2 商品中心（Products）
**核心功能**：
- 商品列表（多维度筛选、排序、批量操作）
- 商品编辑（基本信息、规格参数、图文描述）
- 分类管理（多级分类、分类属性）
- 库存管理（库存预警、补货建议、库存流水）
- 价格管理（价格策略、促销价格、价格历史）
- 批量操作（上下架、改价、改库存、导出）

**特色功能**：
- 智能定价建议（基于竞品数据）
- 库存预测（基于销售趋势）
- 商品体检（标题、主图、详情优化建议）

#### 3.2.3 订单中心（Orders）
**核心功能**：
- 订单列表（多维度筛选、状态跟踪、批量处理）
- 订单详情（商品信息、客户信息、物流信息）
- 发货管理（批量发货、物流跟踪、电子面单）
- 售后管理（退款处理、退货处理、换货处理）
- 评价管理（评价列表、回复管理、差评预警）

**工作流**：
1. 订单接收 → 2. 订单确认 → 3. 备货打包 → 4. 发货通知 → 5. 物流跟踪 → 6. 签收确认 → 7. 售后处理

#### 3.2.4 客户中心（Customers）
**核心功能**：
- 客户列表（多维度筛选、标签管理、分组管理）
- 客户画像（基本信息、消费行为、偏好分析）
- 会员体系（等级设置、积分管理、权益配置）
- 客户分群（RFM模型、生命周期、价值分层）
- 营销触达（短信、微信、Push通知）

**数据分析**：
- 客户增长趋势
- 客户价值分布
- 客户留存分析
- 客户流失预警

#### 3.2.5 营销中心（Marketing）
**核心功能**：
- 优惠券管理（创建、发放、核销、统计）
- 促销活动（满减、折扣、秒杀、拼团）
- 分销管理（分销员、佣金、分销商品）
- 直播管理（直播预约、直播数据、直播回放）
- 内容营销（短视频、图文、直播带货）

**营销工具**：
- 新客专享
- 老客回馈
- 节日营销
- 会员日

#### 3.2.6 数据中心（Analytics）
**核心功能**：
- 销售分析（销售趋势、销售构成、热销商品）
- 客户分析（客户增长、客户价值、客户行为）
- 商品分析（商品表现、库存分析、价格分析）
- 流量分析（流量来源、转化漏斗、页面分析）
- 竞品分析（竞品监控、价格对比、策略分析）

**报表系统**：
- 日报/周报/月报自动生成
- 自定义报表
- 数据导出（Excel、PDF）

#### 3.2.7 客服中心（Service）
**核心功能**：
- 在线客服（多平台接入、智能分配、会话记录）
- 工单管理（工单创建、流转、处理、统计）
- 智能客服（自动回复、意图识别、知识库）
- 客服质检（会话质检、评分、培训）
- 客户满意度（评价收集、满意度分析）

#### 3.2.8 财务中心（Finance）
**核心功能**：
- 收支明细（收入、支出、利润、现金流）
- 提现管理（提现申请、提现记录、手续费）
- 对账管理（平台对账、银行对账、自动对账）
- 发票管理（开票申请、发票记录、税点计算）
- 财务报表（利润表、资产负债表、现金流量表）

#### 3.2.9 供应链（Supply Chain）
**核心功能**：
- 采购管理（采购计划、采购订单、采购入库）
- 供应商管理（供应商档案、评估、合作记录）
- 仓储管理（仓库设置、库存管理、出入库记录）
- 物流管理（物流商管理、物流跟踪、物流费用）

#### 3.2.10 系统管理（System）
**核心功能**：
- 店铺管理（店铺信息、平台授权、店铺设置）
- 权限管理（角色管理、用户管理、权限分配）
- 操作日志（操作记录、异常监控、安全审计）
- 消息通知（系统通知、业务提醒、预警设置）
- 系统设置（基础设置、集成设置、备份恢复）

## 4. UI/UX设计规范

### 4.1 设计原则
- **一致性**：统一的设计语言和交互模式
- **简洁性**：减少视觉噪音，突出核心内容
- **反馈性**：所有操作都有明确反馈
- **容错性**：防止错误操作，提供撤销机制

### 4.2 视觉规范
- **主色调**：#165DFF（蓝色）
- **辅助色**：#F2F3F5（浅灰背景）、#FFFFFF（白色卡片）
- **成功色**：#00B42A（绿色）
- **警告色**：#FF7D00（橙色）
- **错误色**：#F53F3F（红色）
- **圆角**：8px（卡片）、4px（按钮）
- **阴影**：0 2px 8px rgba(0,0,0,0.08)（卡片阴影）

### 4.3 布局规范
- **左侧菜单**：宽度240px，可折叠
- **顶部Header**：高度56px，包含面包屑、搜索、通知、用户信息
- **内容区域**：自适应宽度，内边距24px
- **卡片间距**：16px

### 4.4 组件规范
- **按钮**：主要按钮（蓝色）、次要按钮（边框）、文字按钮
- **表格**：斑马纹、悬停效果、固定表头
- **表单**：标签在上、验证提示、自动保存
- **弹窗**：居中显示、背景遮罩、ESC关闭
